# DCFHYENA

DCFHYENA is a PyTorch-based deep learning framework for transcription factor binding site (TFBS) prediction.

## Highlights

- 🧬 Dual-channel architecture for learning complementary sequence representations.
- 🔗 Novel feature fusion module for effective multi-scale feature integration.
- 🚀 Improved prediction performance compared with existing TFBS prediction models.
- ⚡ Efficient implementation based on the Hyena architecture for long DNA sequences.
